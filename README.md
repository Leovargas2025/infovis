# TP Infovis – Leonardo Vargas

Este repo contiene el sitio para GitHub Pages con:
- `index.html` (estructura con placeholders para iframes)
- `/tp_infovis_assets/*.csv` (datasets listos para subir a RAWGraphs, Flourish, Datawrapper y Tableau)

## Publicación rápida (GitHub Pages)
1. Sube todo el contenido de esta carpeta al repositorio `infovis`.
2. En GitHub, ve a **Settings → Pages**.
3. En **Source**, elige **Deploy from a branch**.
4. Selecciona **main** y la carpeta **/** (root) o **/docs** si prefieres.
5. Guarda. La URL será `https://<tu-usuario>.github.io/infovis/`.

## Inserción de gráficos
1. Crea cada gráfico en la herramienta que quieras.
2. Copia el **iframe** de publicación.
3. Abre `index.html` y pega el iframe donde indica “Pega aquí el iframe del gráfico ...”.
4. Commitea y push. Listo.
